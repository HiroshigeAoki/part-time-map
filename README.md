# part-time-map

タウンワークなどのバイト情報をGoogle Map上にプロットして探しやすくするWebサイトです。
URL: https://part-time-map-355016.web.app/ (今の所、利用可能範囲は静岡大学周辺・富塚エリアのみです（拡大予定）。)
